# PyTorch Tensors and Basic Operations Lab

This repository contains a Jupyter Notebook lab assignment focused on introducing **PyTorch tensors** and performing basic tensor operations.

The notebook provides hands-on practice with tensor creation, manipulation, and mathematical operations using the PyTorch library.

---

## File Included

- **Lab1_5_1_25.ipynb**  
  Covers introductory exercises on PyTorch tensors and their operations.

---

## The Notebook includes : 

- Introduction to PyTorch tensors  
- Creating tensors (zeros, ones, random values)
- Tensor shapes and dimensions
- Indexing and slicing tensors
- Tensor arithmetic operations
- Basic mathematical functions
- Practice questions and solutions
